## Colunas da Tabela de Sugestoes

| Coluna | Descricao | Fonte |
|---|---|---|
| Acao | Ticker + Nome da empresa | Brapi / Realistic |
| Anterior (R$) | Fechamento do dia anterior | Brapi |
| Atual (R$) | Preco atual | Brapi |
| Var. (%) | Variacao percentual | Calculado |
| DY (%) | Dividend Yield 12 meses | Realistic |
| Tipo | JCP ou Dividendo | Regra (bancos = JCP) |
| Valor Pago (R$) | Valor bruto por acao | Calculado |
| Acum. Ano (R$) | Soma YTD de dividendos | Calculado |
| Data Pagto | Proximo pagamento | Realistic |
| Data Corte | Data ex-dividendos | Realistic |
| Noticias | Analise Boa/Ruim/Neutro | Partnr + DeepSeek |
